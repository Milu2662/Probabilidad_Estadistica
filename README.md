## Descripción
Este repositorio contiene la implementación en Python de técnicas de conteo y conceptos básicos de probabilidad, con aplicaciones prácticas en el campo empresarial.

## Contenido

### Técnicas implementadas
- Cálculo de factoriales
- Permutaciones P(n,r)
- Combinaciones C(n,r)
- Probabilidad clásica
- Probabilidad del complemento

### Ejemplos desarrollados

#### Ejemplo 1: Selección de Personal
Problema de formación de comités considerando diferentes perfiles profesionales.

#### Ejemplo 2: Control de Calidad
Análisis probabilístico de muestras en procesos de inspección industrial.

#### Aplicación Empresarial: Optimización de Rutas
Problema real de logística y distribución con análisis de:
- Número de rutas posibles
- Rutas con restricciones
- Modelación probabilística de demanda

## Requisitos
- Python 3.7 o superior
- No se requieren paquetes adicionales (usa solo biblioteca estándar)
